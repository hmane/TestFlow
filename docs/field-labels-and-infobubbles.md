### Preview Section

| Element | Text in Application |
|---------|-------------------|
| Section Title | Workflow Preview |
| Empty State (no selection) | Select a request type above to preview the review workflow. |
| With Selection | Click on each step to view details about what happens at that stage. |

## Notes for Reviewers

1. Fields marked "Yes (non-self-approval)" are required unless the user is in the "LW - Self Approvers" group and selects themselves as the approver.
2. Fields marked "Yes (conditional)" are only required under specific conditions noted in the Required column.
3. RFP submissions (starting with "RFP Related Review") have relaxed validation — approvals, distribution method, and date of first use become optional.
4. Info Bubbles appear as small "i" icons next to the field label. Hovering shows the tooltip text.
5. Placeholder text appears as grayed-out text inside empty input fields.
